New Malware Mines Bitcoin via Victims' CPUs and GPUs

Palo Alto Networks has discovered a unique malware family that can mine bitcoins via the victim's CPUs and GPUs. The malware, called PWOBot, is completely written in Python and compiled via PyInstaller, generating a Microsoft Windows executable, according to a Palo Alto Networks posting. The malware has affected European organizations, mainly in Poland.
